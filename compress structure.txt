│  │                                                                          │   │
│  │  Initial state: 65,536 evenly-spaced boundaries.                        │   │
│  │  Bucket count range: 19,531 (min) — 156,250 (max).                      │   │
│  │  Typical: ~65,000 buckets.                                              │   │
│  │                                                                          │   │
│  │  WHY LZ4 (not zstd):                                                    │   │
│  │    LZ4 decompresses at ~4 GB/s vs zstd ~1 GB/s.                        │   │
│  │    Small buckets (~7KB) → decompression speed is critical.              │   │
│  │    Slightly worse ratio (60% vs 50%) but 4× faster.                    │   │
│  │                                                                          │   │
│  │  Bucket data:                                                           │   │
│  │    ~150 states × 52B avg = ~7.8 KB raw per bucket.                      │   │
│  │    LZ4 compressed: ~4.7 KB (60% ratio).                                │   │
│  │    Total: ~65K buckets × 4.7 KB ≈ ~300 MB data                         │   │
│  │    + extended/overflow + padding + headroom = ~490 MB total.             │   │
│  │                                                                          │   │
│  │  ANTI-ADVERSARIAL PROTECTION:                                            │   │
│  │    • Bucketing uses SipHash-2-4 with secret key (not FNV1a)             │   │
│  │      adversary cannot predict hash distribution                         │   │
│  │    • Max split depth: 256 per region                                    │   │
│  │    • Oversized buckets (>1024 despite splits) get intra-bucket          │   │
│  │      mini-index: 16 evenly-spaced hash checkpoints for fast search     │   │
│  │                                                                          │   │

│                                                                                 │

│  │                                                                          │   │

│  COLD LOOKUP PROCEDURE:                                                         │
│  ┌──────────────────────────────────────────────────────────────────────────┐   │
│  │                                                                          │   │
│  │  cold_lookup(hash64, hash32):                                            │   │
│  │      │                                                                   │   │
│  │      ▼                                                                   │   │
│  │  1. bucket = ring_index.binary_search(hash32)          O(log N)         │   │
│  │  2. if bucket.entry_count == 0: return null                              │   │
│  │  3. compressed = disk_read(bucket.offset, bucket.compressed_size)        │   │
│  │  4. verify CRC32 of compressed data                                     │   │
│  │  5. raw = lz4_decompress(compressed)                    ~2 μs           │   │
│  │  6. entry = binary_search(raw, hash64)                  O(log ~150)     │   │
│  │     (search by 64-bit hash for collision safety)         ~8 steps       │   │
│  │  7a. FOUND → increment access_counter                                   │   │
│  │               if access_counter ≥ 10 → promote to cuckoo overflow       │   │
│  │               return entry                                              │   │
│  │  7b. NOT FOUND → return null (bloom false positive)                     │   │
│  │                                                                          │   │

│  │    Post-move:  bucket may need split if > 512 entries                   │   │

│  │  HYSTERESIS:   promote at ≥ 10, demote at < 5                           │   │

┌─────────────────────────────────────────────────────────────────────────────────┐
│                                                                                 │
│  FUNCTION lookup_state(raw_state_key: bytes) → StateEntry | null                │
│  ┌──────────────────────────────────────────────────────────────────────────┐   │
│  │                                                                          │   │
│  │  // HASH                                                                 │   │
│  │  hash64 = fnv1a_64(raw_state_key)                                        │   │
│  │  hash32 = truncate_32(hash64)                                            │   │
│  │                                                                          │   │
│  │  // BLOOM                                                                │   │
│  │  if NOT bloom.check(hash32):                                             │   │
│  │      return null                                                         │   │
│  │                                                                          │   │
│  │  // TIER 1: MPHF MAIN                                                    │   │
│  │  index = mphf.lookup(hash32)                                             │   │
│  │  if index != INVALID:                                                    │   │
│  │      entry = main_data[offset_table[index]]                              │   │
│  │      if entry.stored_hash_64 == hash64:                                  │   │
│  │          return entry                                                    │   │
│  │                                                                          │   │
│  │  // TIER 2: CUCKOO OVERFLOW                                              │   │
│  │  entry = cuckoo.lookup(hash64)                                           │   │
│  │  if entry != null:                                                       │   │
│  │      return entry                                                        │   │
│  │                                                                          │   │
│  │  // COLD ARCHIVE                                                         │   │
│  │  bucket = cold_ring.find(hash32)                                         │   │
│  │  if bucket.entry_count == 0:                                             │   │
│  │      return null                                                         │   │
│  │                                                                          │   │
│  │  compressed = disk_read(bucket.offset, bucket.compressed_size)           │   │
│  │  if crc32(compressed) != expected:                                       │   │
│  │      raise CorruptionError                                               │   │
│  │                                                                          │   │
│  │  raw = lz4_decompress(compressed)                                        │   │
│  │  entry = binary_search(raw, hash64)                                      │   │
│  │  if entry == null:                                                       │   │
│  │      return null                                                         │   │
│  │                                                                          │   │
│  │  // PROMOTION CHECK                                                      │   │
│  │  bucket.access_counter++                                                 │   │
│  │  if bucket.access_counter >= 10:                                         │   │
│  │      cuckoo.insert(hash64, entry)                                        │   │
│  │      cold_ring.remove(hash64, bucket)                                    │   │
│  │      if bucket.entry_count < 64:                                         │   │
│  │          cold_ring.merge_with_neighbor(bucket)                           │   │
│  │                                                                          │   │
│  │  return entry                                                            │   │
│  │                                                                          │   │
│  └──────────────────────────────────────────────────────────────────────────┘   │

│  ┌──────────────────────────────────────────────────────────────────────────┐   │
│  │                                                                          │   │
│  │  // FIND CASE                                                            │   │
│  │  block_idx = binary_search(case_block_index, case_id)                    │   │
│  │  compressed_block = disk_read(block_idx.offset, block_idx.size)          │   │
│  │  raw_block = zstd_decompress(compressed_block)                           │   │
│  │  case = scan_block(raw_block, case_id)                                   │   │
│  │  if case == null: return null                                            │   │
│  │                                                                          │   │
│  │  // RESOLVE CHAIN                                                        │   │
│  │  if case.chain_pattern == 0:                                             │   │
│  │      // Custom: resolve each hash                                        │   │
│  │      states = [lookup_state_by_hash(h) for h in case.custom_chain]       │   │
│  │  else:                                                                   │   │
│  │      // Pattern: expand template + fill slots                            │   │
│  │      pattern = pattern_dict[case.chain_pattern]                          │   │
│  │      slots = msgpack_decode(case.slot_values)                            │   │
│  │      states = []                                                         │   │
│  │      for hash in pattern.state_sequence:                                 │   │
│  │          if hash & 0xFFFF0000 == 0xFFFF0000:                             │   │
│  │              states.append(SlotValue(slots[hash & 0xFFFF]))              │   │
│  │          else:                                                           │   │
│  │              states.append(lookup_state_by_hash(hash))                   │   │

│  ┌──────────────────────────────────────────────────────────────────────────┐   │
│  │                                                                          │   │
│  │  // ADD TRANSITION                                                       │   │
│  │  new_trans = TransitionEntry(                                            │   │
│  │      target_hash_32 = target_hash32,                                     │   │
│  │      prob_12 = encode_log_prob(prob),                                    │   │
│  │      type_4 = type                                                       │   │
│  │  )                                                                       │   │
│  │  state.transitions.append(new_trans)                                     │   │
│  │                                                                          │   │
│  │  // RENORMALIZE                                                          │   │
│  │  decoded = [decode_log_prob(t.prob_12) for t in state.transitions]       │   │
│  │  sum = Σ decoded                                                         │   │
│  │  if |sum - 1.0| > 0.001:                                                │   │
│  │      scaled = [d / sum for d in decoded]                                 │   │
│  │      new_enc = [encode_log_prob(s) for s in scaled]                      │   │
│  │      // Greedy ±1 step adjustment until within tolerance                │   │
│  │      while |Σ decode(new_enc) - 1.0| > 0.001:                           │   │
│  │          adjust entry with largest rounding error by ±1                  │   │
│  │      for i, t in enumerate(state.transitions):                           │   │
│  │          t.prob_12 = new_enc[i]                                          │   │
│  │                                                                          │   │

 SECTION 9: ENCODING REFERENCE
══════════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────────┐
│                                                                                 │
│  VARINT:                                                                        │
│    Each byte: bit 7 = continue flag, bits 0-6 = 7 payload bits.                │
│    Max 4 bytes = 28 payload bits = 0..268,435,455.                              │
│                                                                                 │
│  LOG-SCALE PROBABILITY (12-bit):                                                │
│    encode(p) = round(-log₂(p) × 256)                                          │
│    decode(v) = 2^(-v / 256)                                                    │
│    stored=0 → 100%, stored=4095 → ~0.0015%                                    │
│                                                                                 │
│  FNV1a-64:                                                                      │
│    offset = 14695981039346656037                                               │
│    prime  = 1099511628211                                                      │
│    for each byte: hash = (hash XOR byte) × prime                               │
│                                                                                 │
│  FNV1a-32 (derived):                                                            │
│    truncate_32(hash64) = hash64 XOR (hash64 >> 32)                             │
│    (fold upper 32 bits into lower for better distribution)                      │
│                                                                                 │
│  SipHash-2-4 (cold archive bucketing):                                          │
│    Keyed PRF with 128-bit secret key.                                          │
│    Used ONLY for bucket assignment (adversarial resistance).                   │
│    Key rotated periodically (triggers full re-bucketing).                       │
│                                                                                 │
│  MSGPACK:                                                                       │
│    Binary serialization, self-describing.                                      │
│    fixstr: 0xa0-0xbf + bytes (strings ≤ 31 bytes, 1B overhead).               │
│    fixarray: 0x90-0x9f + elements (arrays ≤ 15, 1B overhead).                  │
│                                                                                 │
│  SLOT SENTINEL:                                                                 │
│    Hash values 0xFFFF0000..0xFFFFFFFF reserved for slot markers.               │
│    Lower 16 bits = slot index (supports up to 65,536 slots/pattern).           │
│    These values excluded from MPHF key set. Collision: negligible.             │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
